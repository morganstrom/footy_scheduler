# Claude Code Context

This file ensures important project guidelines are always available to Claude Code.

## Project Guidelines

Please review and follow the instructions in [AGENTS.md](../AGENTS.md) for all development tasks in this repository.

Key points:
- Separate domain logic (`main.ts`) from UI code (`app.ts`)
- Follow TypeScript strict mode and naming conventions
- Write tests for new features
- Run `npm run build` and `npm test` before committing
- Use imperative commit messages

For detailed instructions, always refer to the full AGENTS.md file.
